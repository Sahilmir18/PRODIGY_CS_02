## Pixel Manipulation for Image Encryption
Building a simple Image encryption tool using pixel manipulation like swapping pixel values or applying a basic mathematical operation to each pixel.
Allow users to encrypt and decrypt images

## How to use
1. Download the main python file main.py
2. Run the file
3. choose the key
4. Insert the Image and Encrypt/Decrypt it
5. Save the file
